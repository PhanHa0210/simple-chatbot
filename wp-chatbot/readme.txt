Chatbot tương tác với người dùng thông qua webhook n8n.

== Description ==

WP Chatbot là plugin giúp bạn thêm một chatbot vào website WordPress của mình. Chatbot này kết nối với webhook n8n để xử lý các câu hỏi của người dùng.

Tính năng chính:
* Hiển thị nút chat ở góc dưới bên phải màn hình
* Kết nối với webhook n8n để xử lý tin nhắn
* Tùy chỉnh màu sắc chủ đạo
* Tùy chỉnh tin nhắn chào mừng
* Responsive trên mọi thiết bị

== Installation ==

1. Tải plugin lên thư mục `/wp-content/plugins/`
2. Kích hoạt plugin qua menu 'Plugins' trong WordPress
3. Vào 'WP Chatbot' trong menu admin để cấu hình

== Frequently Asked Questions ==

= Làm thế nào để kết nối với webhook n8n? =

Bạn cần có một webhook URL từ n8n. Nhập URL này vào phần cài đặt của plugin.

= Làm thế nào để thay đổi màu sắc của chatbot? =

Vào phần cài đặt của plugin và chọn màu sắc chủ đạo mong muốn.
